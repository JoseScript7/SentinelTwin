# SentinelTwin

Inventory forecasting and disaster-resilient supply chain management for emergency response and routing workflows.

## What’s in this repo

- `gateway-server/` contains the Node.js backend, dashboard apps, ML pipeline, and project documentation.
- `mobile-app/` contains the mobile SOS application shell.

## Start locally

1. Install dependencies in `gateway-server/` if needed.
2. Run the gateway server from `gateway-server/server.js`.
3. Open the dashboard or mobile app entry points from the `gateway-server/dashboard/` and `mobile-app/` folders.
